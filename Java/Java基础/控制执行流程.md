# 控制执行流程

## 条件语句

- if-else语句

  ```java
    if(Boolean-expression)
    	statement
  
    if(Boolean-expression)
    	statement
    else
      statement
  
    if(Boolean-expression)
    	statement
    else if(Boolean-expression)
      statement
    else
      statement
  ```

- switch语句

  ```java
    switch(integral-selector){
      case integral-value1:statement;break;
    	case integral-value2:statement;break;
    	case integral-value3:statement;break;
      default:statement;
    }
  ```

## 循环语句

- while语句

  ```java
    while(Boolean-expression)
    	statement
  ```

- do-while语句

  ```java
    do
    	statement
    while(Boolean-expression)
  ```

- for语句

  ```java
    for(initialization;Boolean-expression;step)
    	statement
  ```

- ForEach语句

  ```java
    for(item:arrays)
    	statement
  ```

## 关键字

- `break` - 退出循环语句
- `continue` - 退出当前循环
