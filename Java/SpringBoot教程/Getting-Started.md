# Getting Started
Spring Boot是基于Java的开源框架，可以轻松创建独立的，生产级的基于Spring的应用程序
- 创建独立的Spring应用程序
- 直接嵌入Tomcat，Jetty或Undertow
- 提供默认依赖项以简化构建配置
- 尽可能自动配置Spring和第三方库
- 提供生产就绪功能，例如指标，运行状况检查和外部化配置
